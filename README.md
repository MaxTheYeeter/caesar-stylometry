# Caesar Stylometry Project

A computational stylometric investigation of the composition history of Julius Caesar's *De Bello Gallico*.

## Research Question

Were Books I–VII written incrementally during the campaigns or composed retrospectively near the end of the Gallic War?

## Validation

Book VIII, known to have been written by Aulus Hirtius, serves as a validation test for authorship detection methods.

## Tools

- Python
- CLTK
- Scikit-learn
- R
- Stylo
